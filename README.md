# Villary

A luxury villa booking platform built with React, TypeScript, and Vite.

## Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

## Features

- Luxury villa browsing and booking
- User authentication
- Room selection and checkout
- Interactive concierge chat interface
- Responsive design with Tailwind CSS

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS (CDN)
- Lucide React Icons
